Requirements

  • A web browser to view HTML/CSS/JS code
  • A web server/host to host HTML/CSS/JS if you plan on hosting this code.
  • An SCSS watcher if you want to use the SCSS file (completely optional)

Setup

  1. Place the pricingtables folder onto your web server
  2. View the project and all pricing tables within by loading index.html.
  3. To see the code and how to use a pricing table, load that specific table e.g. pt.card.html.
